The best Tool: SET

NCSA Storage Group

Well over 100 years cumulative experience

Experience on a wide variety of challenging transfer projects

Minio

S3-based object server (over posix FS) - high performance

Supports clustered erasure-code storage for reliability

Single-file deploy, no admin requirement

Web interface, CLI, and API, integrates with OAuth/JWT

High performance, modern multicore implementation

Tools: Globus

Service-oriented, "managed" transfers

Retries failed transfers

Requires admin at each endpoint, nontrivial setup

Web, CLI, API(python), queryable transfer state, limited logging

Performance issues with large #'s of small files

Tools: GIT

Merkel-tree based versioned distributed file sharing

Differential updates, multi-site sharing

Good up to ~10GB of text-based data

Optional new line protocol is very efficient

CLI, ssh-based auth and transport

Tools: rsync

Service and/or ad-hoc transfers, highly configurable

Can use ssh transport/keys

Copy or sync, push or pull

Supports reliability, differential updates

CLI only, for fine-grained logging
